01

About the Project

AlgoViz is a web-based algorithm visualization platform that allows users to watch sorting, searching, and graph algorithms execute step by step in real time. Rather than reading static pseudocode or tracing through diagrams by hand, users can interact with live animations, observe code highlighting, and read plain-English explanations at every stage of execution.

The platform is built on a clean separation of concerns: a Python Flask backend handles algorithm computation and generates structured step data, while a Vanilla JavaScript frontend handles animation, rendering, and user interaction. This architecture makes the system easy to extend — adding a new algorithm only requires writing one Python function and registering it on the frontend.

02

Purpose

The primary goal of AlgoViz is to bridge the gap between algorithmic theory and intuitive understanding. Students and beginners learning data structures and algorithms often struggle with the abstract nature of sorting passes, pointer movements, and graph traversals. Textbooks describe what an algorithm does; AlgoViz shows exactly how it does it, one step at a time.

By watching Bubble Sort swap elements, or Binary Search eliminate half the array with each comparison, learners develop a mental model that persists far longer than anything read on a page. AlgoViz is designed as a tool for both self-directed learners and university-level computer science education.

03

Key Features

AlgoViz provides a full set of interactive tools designed to maximize learning at every level:

⚡

Real-Time Animation

Every array state change, swap, comparison, and pointer move animates fluidly in the browser with smooth CSS transitions.

📄

Live Code Highlighting

The Python source code is displayed alongside the animation, with the active line highlighted at every step.

📊

Sorting Algorithms

Bubble Sort, Insertion Sort, Selection Sort, Merge Sort, and Quick Sort — all fully animated with detailed step logic.

🔍

Searching Algorithms

Linear Search and Binary Search with custom target inputs. Watch how binary search's O(log n) efficiency plays out visually.

🕸️

Graph Algorithms

BFS, DFS, and Dijkstra visualized on an interactive node graph with traversal order shown in real time.

🎛️

Full Playback Control

Start, Pause, Step-forward, and Reset controls with a speed slider from 1x to 10x. Review any step as many times as needed.

✏️

Custom Input Arrays

Enter any comma-separated array, generate a random one, or use the default preset to explore edge cases.

📖

Step Explanations

Every step includes a plain-English description of what happened and why — not just what the code did.

05

Educational Value

Research in cognitive science consistently shows that visual representations of abstract processes significantly improve comprehension and retention. Algorithm visualization makes complexity tangible: when students can see that Merge Sort always takes O(n log n) steps while Bubble Sort degrades toward O(n²), the runtime analysis moves from abstract notation into lived, observable experience.

AlgoViz is particularly valuable for students preparing for technical interviews, computer science coursework, and competitive programming. Understanding algorithms at an intuitive level — not just at a syntactic level — is the difference between being able to recall an algorithm and being able to reason through variations of it independently.

The side-by-side layout of code, animation, and explanation is designed to build three complementary mental models simultaneously: the implementation, the data transformation, and the conceptual logic.
